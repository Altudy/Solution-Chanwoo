2. D번- 수학적으로 f(x+B) = f(x) 즉, 반복된다는 사실을 알면 x가 [0, B-1] 구간일 때만을 고려하여 계산할 수 있다. \
   식을 정리하면, f(x) = floor(Ax/B) 로 x에 대해 monotonically non-decreasing한 함수이므로, x=B-1일 때 답이 나옴을 알 수 있다.
